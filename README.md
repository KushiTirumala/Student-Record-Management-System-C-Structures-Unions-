A console-based C program that manages student records using structures and unions. Users can view, update, and display student information in a simple menu-driven interface.

Project Overview

This project demonstrates the use of C programming structures for organizing student data and unions for handling different types of marks efficiently. The system is menu-driven, allowing users to interactively manage student records.

Features

Store student information: ID, Name, Age, and Marks

View all student records

Display a specific student by ID

Update student marks

Efficient memory usage using unions for marks

Technologies Used

C Programming Language

Structures for organizing student data

Unions for memory-efficient storage of marks

Console input/output

How to Run

Clone the repository:

git clone https://github.com/your-username/student-record-c.git


Navigate to the project folder:

cd student-record-c


Compile the program:

gcc student_record.c -o student_record


Run the executable:

./student_record


Follow the menu to view, update, or manage student records
